## 🕹 Features
- SNS 로그인 (kakao, google)
- 종목 순위 정보(실시간, 등락율, 시가총액)
- 종목 상세 정보 및 차트 제공
- 주가 예측
- 종목 검색
- 종목 즐겨찾기



### Tech Stack & Library
- MVC
- retrofit2, glide
- MPandroidchart
